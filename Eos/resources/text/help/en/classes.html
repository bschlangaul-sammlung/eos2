Classes

You can define your own classes. It allows to define your own structures of data, allowing to create a group of variables at once. In addition you can can you own methods. You also can inherit existing classes an extend them.

Syntax

Einfach Klasse

class Name
  Properties
  Methods
*classe

Class with super class

class Name extends Name
  Properties
  Methods
*classe

BNF

UserClass :=
   "class" Name ( "extends" Name )? ( Method | Property)* "*class"
Method := A method
Property := a normal Variable declaration
